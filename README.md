## 소셜위드
```
  ### 프로젝트 설명
  공공기관들을 대상으로 여성기업, 장애인 기업 등 사회적 기업의 MRO 상품을 구매할 수 있는 B2G 쇼핑몰 구축

  ** 게시판의 기본 기능 **<br>
  C : Create<br>
  R : Read<br>
  U : Update<br>
  D : delete<br>

  ### 사용언어
  ASP.NET / C# / JavaScript / jquery1.10.2 / Oracle 11g

  ### 직무 내용
  사이트 내 UI 고도화 개발 및 DB 쿼리 수정

  ### 기여도 및 참여도
  총 5M 투입 
```
